# PINN for DT4LC project
This is the main test implementation of the PINN model, which implements integro-difference PINN architecture for precipitation prediction.
Core of the PINN component ofr DT4LC project.

Run:
```bash
git clone https://github.com/IPT-MMDA/PINN-DT4LC-Model
cd PINN-DT4LC-Model/
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
python app.py
```
